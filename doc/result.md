# 評価

提案手法と提案手法から工夫を取り除いた手法の操作回数を比較することで、適用した工夫の有効性をら評価する。
また、最も工夫が有効に働いたテストデータでの各手法の挙動を確認することで、工夫が期待通りの効果をもたらしているかを確認する。

## 実験条件

テストデータセット：Web 版ビジュアライザのデフォルト設定(seed 0,サイズ 100,N 及び M についての指定なし)
